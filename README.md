**This repo is no longer maintained, please use vue-cli instead**

A simple Vue + Webpack yeoman generator

**NOTE:** It only supports Vue 1.0.x now

## Installation && Usage 
```
$ npm install -g generator-vuejs
$ mkdir myproject && cd myproject
$ yo vuejs
```

## Command
```
npm run dev
npm run build	
```
